互動式探索：矩陣 ATA 的可逆性

一個關於線性代數中重要定理的直觀指南

核心問題與答案

給定一個 n × m 的矩陣 A，其衍生的 m × m 方陣 B = ATA 何時為可逆矩陣？

答案：若且唯若，矩陣 A 的所有行向量（columns）線性獨立。

證明的邏輯鏈

這個定理的證明基於一系列優雅的等價關係。下面的流程圖展示了整個邏輯推導的關鍵步驟，將 ATA 的可逆性與 A 的性質聯繫起來。

關鍵概念解析

一個 m × m 的方陣 B 可逆，如果存在另一個矩陣 B-1 使得 BB-1 = B-1B = I。這有多個等價條件：

  • 其秩等於其維度 m (滿秩)。
  • 其核空間 N(B) 只包含零向量。
  • 其所有行向量（或列向量）線性獨立。
  • 其行列式不為零 det(B) ≠ 0。
  • 齊次方程組 Bx = 0 只有唯一解 x = 0。

這是證明中最巧妙的部分，需要雙向證明：

證明 N(A) ⊆ N(ATA):

若 x 在 N(A) 中，則 Ax = 0。用 AT 左乘，得 AT(Ax) = AT0，即 ATAx = 0。所以 x 也在 N(ATA) 中。

證明 N(ATA) ⊆ N(A):

若 x 在 N(ATA) 中，則 ATAx = 0。用 xT 左乘，得 xTATAx = 0。這可以寫成 (Ax)T(Ax) = 0，也就是向量 Ax 的長度平方 ||Ax||2 = 0。一個向量長度為零，若且唯若該向量本身是零向量。因此 Ax = 0，所以 x 也在 N(A) 中。

既然兩個集合互為子集，它們必然相等。

維度約束：一個必要的先決條件

為了使矩陣 A 的 m 個行向量線性獨立，A 的維度必須滿足一個基本條件：行數 n 必須大於或等於列數 m (n ≥ m)。請使用下方的滑桿來探索這個概念。

應用：最小平方法

這個定理在數據科學中至關重要。當我們想用一條直線擬合一組數據點時（線性迴歸），我們會求解一個線性系統 Ax = b。通常這個系統沒有精確解，我們會尋找一個「最佳擬合」解，這可以通過求解正規方程 ATAx = ATb 來找到。如果 ATA 可逆，我們就能找到唯一的最佳解 ◯ = (ATA)-1ATb。
